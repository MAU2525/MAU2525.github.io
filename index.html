Definiciones:

HTML (Hyper Text Markup Languaje):

Es el lenguaje de marcado que define la estructura
y el contenido de una pagina web. Utiliza etiquetas para delimitar diferentes
elementos, como parrafos, encabezados, imagenes, enlaces, etc.

CSS (Cascading Style Sheets):

Es un lenguaje de estilos que se utiliza para controlar
la presentacion de una pagina web. Permite definir aspectos como los
colores, fuentes, tama単os, margenes, etc., de los elementos HTML.

JavaScript ( Program. Language):

Es un lenguaje de programacion que permite agregar funcionalidad
interactiva a una pagina web. Permite manipular elementos HTML, responder a
eventos del usuario, realizar calculos, etc.
